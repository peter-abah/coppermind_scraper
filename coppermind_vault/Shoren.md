**Shoren** is an ardent on the [[Shattered Plains]]. 

[[Adolin Kholin]] uses Shoren's scribing services to write to [[Shallan Davar]]. Since Adolin makes a point to mention to Shallan that he was using an ardent instead of the other Kholin scribes, Shallan wonders if Adolin would worry Shallan would grow jealous if he used a woman instead for scribing.
