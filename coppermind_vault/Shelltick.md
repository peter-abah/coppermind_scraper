**Shelltick** is a substance on [[Roshar]] that can be crushed and used as livestock feed. It has an unknown property that can help inhibit the second pupation of adult [[chull]]s if they eat it before a [[highstorm]]. Shelltick is likely organic, but it is unclear if it is a plant, an animal, or something else entirely.
