***Infinity Blade: Redemption*** is a novella by [[Brandon Sanderson]] that is a tie-in to an iOS game, [[Infinity Blade]] (made by [[Wikipedia:Epic Games|Epic Games]]). *Redemption* is set between Infinity Blade 2 and Infinity Blade 3. It is set in the same universe as [[Infinity Blade: Awakening]].
